# 1. Introducción del producto

![](media/0eed85e053eec9481023a1aaa999190e.png)

## 1.1 Introducción

Este robot tanque educativo STEM V3.0 ha sido recientemente actualizado, añadiendo una función de seguimiento de línea y una función de extinción de incendios. Mejora vigorosamente la relación entre niños y padres, y despierta la imaginación de los niños a través de la programación y la codificación.

Durante el proceso de montaje, podrá observar sus múltiples funciones como seguimiento de luz, seguimiento de línea, control remoto IR y BT, ajuste de velocidad, etc. Además, hay algunas piezas pequeñas que pueden ayudarle a montar el coche robot.

Se incluyen sensores y módulos básicos, como un sensor de llama, un sensor BT, un sensor de evitación de obstáculos, un sensor de seguimiento de línea y un sensor ultrasónico.

Los dos tutoriales para el código en lenguaje C del IDE de Arduino y la programación gráfica de KidsBlock también son adecuados para entusiastas de diferentes edades.

Es realmente la mejor opción para usted.

## 1.2 Características

1. Múltiples funciones: Confinamiento, seguimiento de línea, extinción de incendios, seguimiento de luz, control remoto IR y BT, control de velocidad, etc.

2. Fácil de construir: ensamble el robot con algunas piezas.

3. Alta tenacidad: Soportes de aleación de aluminio, motores metálicos, ruedas de alta calidad.

4. Alta extensión: conecte muchos sensores y módulos a través del escudo del controlador de motor y piezas LEGO.

5. Múltiples controles: control remoto IR, control por aplicación (sistemas iOS y Android).

6. Programación básica: código en lenguaje C del IDE de Arduino y programación gráfica de KidsBlock.

## 1.3 Parámetros

- Voltaje de trabajo: 5V

- Voltaje de entrada: 6-9V

- Corriente máxima de salida: 1.5A

- Disipación máxima de potencia: 32W

- Velocidad del motor: 5V 200 rpm / min

- Modo de accionamiento del motor: accionamiento de doble puente H (HR8833)

- Ángulo de inducción ultrasónica: <15°

- Distancia de detección ultrasónica: 2cm-300cm

- Distancia de control remoto infrarrojo: 10 metros (medida)

- Distancia de control remoto BT: 30 metros (medida)

## 1.5 Placa de desarrollo Keyestudio V4.0

Debe saber que la placa de desarrollo Keyestudio V4.0 es el núcleo de este coche inteligente.

![](./media/image-20250709102941963.png)

La placa de desarrollo Keyestudio V4.0 se basa en el microcontrolador ATmega328P y utiliza un chip CP2102 como convertidor UART a USB.

![](./media/image-20250709103216112.png)

Tiene 14 pines de entrada/salida digitales (de los cuales 6 pueden usarse como salidas PWM), 6 entradas analógicas, un cristal de cuarzo de 16 MHz, una conexión USB, un conector de alimentación, 2 cabezales ICSP y un botón de reinicio.

![](./media/image-20250709103241560.png)

Podemos alimentarlo con un cable USB, el conector de alimentación de CC externo (CC 7-12V) o los pines hembra Vin/GND (CC 7-12V).

| Microcontrolador | ATmega328P-PU |
| :-------------------------: | :------------------------------------------------------: |
| Voltaje de funcionamiento | 5V |
| Voltaje de entrada (recomendado) | CC7-12V |
| Pines de E/S digitales | 14 (D0-D13) (de los cuales 6 proporcionan salida PWM) |
| Pines de E/S digitales PWM | 6 (D3, D5, D6, D9, D10, D11) |
| Pines de entrada analógica | 6 (A0-A5) |
| Corriente CC por pin de E/S | 20 mA |
| Corriente CC para pin de 3.3V | 50 mA |
| Memoria Flash | 32 KB (ATmega328P-PU) de los cuales 0.5 KB son utilizados por el gestor de arranque |
| SRAM | 2 KB (ATmega328P-PU) |
| EEPROM | 1 KB (ATmega328P-PU) |
| Velocidad de reloj | 16 MHz |
| LED_BUILTIN | D13 |
